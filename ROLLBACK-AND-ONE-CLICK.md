# Точка отката и запуск одной кнопкой

## Что зафиксировано

Это состояние проекта зафиксировано как новая точка отката после добавления:
- Dockerfile
- docker-compose.yml
- .dockerignore
- запуска через `start-hormuz-docker.bat`
- остановки через `stop-hormuz-docker.bat`

## Как запускать одной кнопкой на Windows 10

Сделайте двойной клик по файлу:

- `start-hormuz-docker.bat`

Что произойдет:
- запустится Docker Compose
- соберётся контейнер приложения
- поднимутся Redis и Postgres
- через несколько секунд откроется `http://localhost:5000`

## Что нужно перед первым запуском

- установить Docker Desktop
- запустить Docker Desktop
- дождаться статуса Engine running
- убедиться, что рядом лежит файл `.env.production`

## Как остановить

Сделайте двойной клик по файлу:

- `stop-hormuz-docker.bat`

## Как откатиться к этой точке

Если захотите вернуться именно к этому состоянию:
- удалите текущую рабочую папку проекта или переименуйте её
- распакуйте архив отката в новую папку
- положите рядом `.env.production`
- сделайте двойной клик по `start-hormuz-docker.bat`

## Что важно помнить

Сейчас приложение действительно использует SQLite, сохранённый в Docker volume.

Это значит:
- карта и backend работают сразу
- JWT и реальный AIS-поток работают сразу
- Postgres и Redis уже входят в стек, но будут использоваться кодом полностью после отдельной миграции backend
